# File Handling Utility in Java

## Internship Task 1

This project demonstrates file handling operations in Java including:
- Reading from a text file
- Writing to a text file
- Modifying (appending) file content

## Technologies Used
- Java
- FileReader / FileWriter
- BufferedReader / BufferedWriter

## How to Run
1. Clone the repository
2. Open the project in any Java IDE (VS Code / IntelliJ / Eclipse)
3. Run `FileHandlingUtility.java`
4. Output will be shown in the console and file will be updated
